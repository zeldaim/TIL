# \# Nmap 포트 스캔 결과 분석

# 

# \## 스캔 대상

# \- 공격자: Kali Linux (192.168.190.130)

# \- 대상: Rocky Linux (192.168.190.128)

# 

# \## 명령어

# nmap -sV -Pn 192.168.190.128

# 

# \## 결과

# | 포트 | 상태 | 서비스 |

# |------|------|--------|

# | 22   | open | SSH OpenSSH 8.7 |

# | 443  | open | Wazuh Dashboard |

# | 9200 | open | Wazuh Indexer |

# | 9090 | closed | - |

# 

# \## 분석

# 443 포트 → Wazuh Dashboard 확인됨

# 9200 포트 → OpenSearch (Wazuh Indexer) 확인됨

# 22 포트 → SSH 열려있음 ← 브루트포스 공격 대상

#

\## 배운 것

\- Kali가 Rocky Linux 스캔해서

&#x20; SSH 포트 22 열린 것 확인

\- 같은 네트워크 상에 있어야 스캔 가능

\- -Pn 옵션: ping 없이 강제 스캔

\- -sV 옵션: 서비스 버전까지 탐지
