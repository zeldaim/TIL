"9201:9200"
 ↑     ↑
호스트  컨테이너

"9201:9200"
→ 호스트(Rocky Linux)의 9201 포트로 접속하면
→ 컨테이너 안의 9200 포트로 연결해줘
컨테이너 안의 9200은 그대로 유지해야 해요
→ Elasticsearch는 내부적으로 항상 9200 사용
→ 컨테이너 내부 설정 건드리면 복잡해짐

그래서 호스트 포트만 9201로 바꿔서
→ 외부에서 9201로 접속하면
→ 컨테이너 9200으로 연결되는 구조
